# 03 — Desarrollo Frontend

> Frontend = lo que el usuario VE y TOCA en la pantalla.

## 🛣️ Ruta recomendada

```
HTML  →  CSS  →  JavaScript  →  Git  →  React  →  TypeScript  →  Next.js
```

---

## 1️⃣ HTML — La estructura

- 📺 [HTML completo - midudev](https://www.youtube.com/@midudev/search?query=HTML+completo)
- 📖 [MDN HTML](https://developer.mozilla.org/es/docs/Web/HTML)
- 🎯 **Proyecto:** Replicá una página simple (ej: tu CV en HTML)

---

## 2️⃣ CSS — Los estilos

- 📺 [CSS desde cero - Bluuweb](https://www.youtube.com/@bluuweb/search?query=CSS+desde+cero)
- 📺 [Flexbox - Carlos Azaustre](https://www.youtube.com/@CarlosAzaustre/search?query=Flexbox)
- 🎮 [Flexbox Froggy](https://flexboxfroggy.com/#es)
- 🎮 [Grid Garden](https://cssgridgarden.com/#es)
- 🎨 [CSS Battle](https://cssbattle.dev/) — retos de CSS

### Frameworks CSS
- **Tailwind CSS** — utility-first, el más usado en 2026.
  - 📺 [Curso Tailwind - midudev](https://www.youtube.com/@midudev/search?query=Curso+Tailwind)
  - 📖 [tailwindcss.com](https://tailwindcss.com/docs)
- **Shadcn/ui** — componentes pre-armados sobre Tailwind.

## 3️⃣ JavaScript en el navegador

- Events (clicks, teclado, etc.)
- Fetch API (traer datos de internet)

## 4️⃣ React — La librería más popular

**Por qué React:** Lo usan Meta, Netflix, Airbnb. Ofertas laborales gigantes.

### Recursos
- 📺 [React desde cero - midudev](https://www.youtube.com/watch?v=7iobxzd_2wY)
- 📺 [Curso React - Fazt Code](https://www.youtube.com/watch?v=DLX62G4lc44)
- 📖 [React.dev (docs oficiales nuevas)](https://react.dev/)
- 📖 [Tutorial Tic-Tac-Toe oficial](https://react.dev/learn/tutorial-tic-tac-toe)

### Conceptos clave
- Componentes
- Props
- State (`useState`)
- Effects (`useEffect`)
- Hooks customs
