```

Auth:        Clerk o Better Auth
Hosting:     Vercel
Payments:    Stripe
Emails:      Resend
AI:          Anthropic (Claude) + Vercel AI SDK
Monitoring:  Sentry + PostHog
```

Es **boring**, pero **funciona**.
