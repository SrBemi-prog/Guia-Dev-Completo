## 🎯 ¿Cómo usar?

1. Forkeá el repo a tu GitHub
2. Edití este archivo (cambiá `[ ]` por `[x]` al completar)
3. Commiteá cada semana — vas a ver tu progreso en el historial
4. Tu GitHub se vuelve tu **timeline real de aprendizaje**
